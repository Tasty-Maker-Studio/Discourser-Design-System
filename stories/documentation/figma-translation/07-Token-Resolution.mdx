# Token Resolution Reference

Complete reference for every token layer in the DDS — from Figma variable to Panda CSS usage.
All values sourced from `src/languages/material3.language.ts` and `dist/figma-variables.json`.

---

## Token System Architecture

```
Layer 1 — Primitives  (raw tonal palette)
  primary/0 … primary/100, secondary/0 … secondary/100, neutral/0 … neutral/100, etc.
  Source: Figma "Primitives" collection · src/languages/material3.language.ts
  Rule: Never use in component code. These are the raw paint, not the meaning.

        ↓ semantic roles are assigned from Primitives

Layer 2 — Semantic  (M3 color roles, mode-aware)
  primary, onSurface, surface/container, outline/variant, background, …
  Source: Figma "Semantic" collection (Light / Dark modes)
  Rule: Always prefer Semantic tokens in component code. They swap automatically in dark mode.

        ↓ Panda CSS consumes Semantic tokens and exposes them as CSS custom properties

Layer 3 — Panda CSS  (design props on components and boxes)
  bg="primary"  color="onSurface"  borderColor="outline.variant"  p="md"  rounded="sm"
  Source: styled-system/tokens  (generated from panda.config.ts)
  Rule: These are what you write. Look up the Semantic token, use its Panda CSS path.
```

## Spacing Scale

Source: `Figma "Spacing & Shape"` collection · `dist/figma-variables.json`

| Figma token | Value | Panda CSS | Typical use |
|---|---|---|---|
| `spacing/none` | `0px` | `p="0"` / `gap="0"` | Reset, flush layouts |
| `spacing/xxs` | `2px` | `p="xxs"` | Icon-to-label nudge, hairline gap |
| `spacing/xs` | `4px` | `p="xs"` | Tight internal component padding |
| `spacing/sm` | `8px` | `p="sm"` | Icon padding, chip internal gap |
| `spacing/2xs` | `12px` | `p="2xs"` | Between-element gap (compact density) |
| `spacing/md` | `16px` | `p="md"` | **Default padding, card content** |
| `spacing/lg` | `24px` | `p="lg"` | Section padding, dialog padding |
| `spacing/xl` | `32px` | `p="xl"` | Panel padding, large card padding |
| `spacing/xxl` | `48px` | `p="xxl"` | Page-level section spacing |
| `spacing/xxxl` | `64px` | `p="xxxl"` | Hero / top-of-page padding |

```tsx
// Correct: use named tokens, not raw pixel values
<Box p="md" gap="sm">...</Box>

// Wrong: hardcoded pixels — doesn't track with spacing scale changes
<Box style={{ padding: '16px', gap: '8px' }}>...</Box>
```

## Elevation (Shadow) Levels

Source: `dist/figma-effect-styles.json`

| Level | Panda CSS | Shadow value | Use case |
|---|---|---|---|
| `level0` | `shadow="level0"` | `none` | Flat surfaces, inline content |
| `level1` | `shadow="level1"` | `0px 1px 2px rgba(0,0,0,0.3), 0px 1px 3px 1px rgba(0,0,0,0.15)` | Cards at rest, contained buttons |
| `level2` | `shadow="level2"` | `0px 1px 2px rgba(0,0,0,0.3), 0px 2px 6px 2px rgba(0,0,0,0.15)` | Cards on hover, raised buttons |
| `level3` | `shadow="level3"` | `0px 4px 8px 3px rgba(0,0,0,0.15), 0px 1px 3px rgba(0,0,0,0.3)` | **Dialogs, dropdowns, popovers** |
| `level4` | `shadow="level4"` | `0px 6px 10px 4px rgba(0,0,0,0.15), 0px 2px 3px rgba(0,0,0,0.3)` | Navigation drawers, modal sheets |
| `level5` | `shadow="level5"` | `0px 8px 12px 6px rgba(0,0,0,0.15), 0px 4px 4px rgba(0,0,0,0.3)` | FABs, tooltips, snackbars |

```tsx
// Correct
<Box shadow="level3">Dialog content</Box>

// Wrong: copy-pasting shadow string directly
<Box boxShadow="0px 4px 8px 3px rgba(0,0,0,0.15)...">...</Box>
```

## Typography Scale

Source: `dist/figma-text-styles.json` · `src/languages/material3.language.ts`

Two font families in use:
- **Fraunces** — Display and headline styles (serif, expressive)
- **Poppins** — Body, title, and label styles (sans-serif, readable)

| Style | Family | Size | Weight | Line height | Use for |
|---|---|---|---|---|---|
| `displayLarge` | Fraunces | 57px | 400 | 64px | Hero headlines, splash screens |
| `displayMedium` | Fraunces | 45px | 400 | 52px | Section hero text |
| `displaySmall` | Fraunces | 36px | 400 | 44px | Large feature text |
| `headlineLarge` | Fraunces | 32px | 400 | 40px | Page titles, modal headers |
| `headlineMedium` | Fraunces | 28px | 400 | 36px | Card titles, panel headers |
| `headlineSmall` | Fraunces | 24px | 400 | 32px | Section subheadings |
| `titleLarge` | Poppins | 22px | 500 | 28px | Dialog titles, list section labels |
| `titleMedium` | Poppins | 16px | 500 | 24px | **Component titles, form labels** |
| `titleSmall` | Poppins | 14px | 500 | 20px | Overlines, small labels |
| `bodyLarge` | Poppins | 18px | 400 | 28px | **Body copy, paragraph text** *(bumped from M3 default 16px)* |
| `bodyMedium` | Poppins | 14px | 400 | 20px | Secondary body text, helper text |
| `bodySmall` | Poppins | 12px | 400 | 16px | Caption text, timestamps |

## Motion Scale

Source: `src/languages/material3.language.ts` · `dist/figma-variables.json`

### Durations

| Token | Value | Panda CSS | Use for |
|---|---|---|---|
| `duration/instant` | `0ms` | `transitionDuration="instant"` | No-animation mode, accessibility override |
| `duration/fast` | `100ms` | `transitionDuration="fast"` | Micro-interactions: button press, icon swap |
| `duration/normal` | `200ms` | `transitionDuration="normal"` | **Default: hover states, input focus, badges** |
| `duration/slow` | `300ms` | `transitionDuration="slow"` | Panel slides, dialogs opening |
| `duration/slower` | `500ms` | `transitionDuration="slower"` | Page transitions, onboarding animations |

### Easing curves

| Name | Curve | Use for |
|---|---|---|
| `standard` | `cubic-bezier(0.2, 0, 0, 1)` | Most transitions — smooth deceleration |
| `standardDecelerate` | `cubic-bezier(0, 0, 0, 1)` | Elements entering the screen |
| `standardAccelerate` | `cubic-bezier(0.3, 0, 1, 1)` | Elements leaving the screen |
| `emphasized` | `cubic-bezier(0.2, 0, 0, 1)` | High-attention transitions (same as standard) |
| `emphasizedDecelerate` | `cubic-bezier(0.05, 0.7, 0.1, 1)` | Hero elements entering |
| `emphasizedAccelerate` | `cubic-bezier(0.3, 0, 0.8, 0.15)` | Hero elements leaving |

```tsx
// Correct: named tokens
<Box transition="all" transitionDuration="normal" transitionTimingFunction="standard">
  Animated element
</Box>
```

## Quick Decision Guide

### Text

| Purpose | Token | Panda CSS |
|---|---|---|
| Body / paragraph text | `onSurface` | `color="onSurface"` |
| Secondary / helper text | `onSurface/variant` | `color="onSurface.variant"` |
| Text on primary button | `onPrimary` | `color="onPrimary"` |
| Text on secondary button | `onSecondary` | `color="onSecondary"` |
| Text on page background | `onBackground` | `color="onBackground"` |
| Error message text | `onError/container` | `color="onError.container"` |
| Tooltip / snackbar text | `inverseOnSurface` | `color="inverseOnSurface"` |

### Backgrounds

| Purpose | Token | Panda CSS |
|---|---|---|
| Page background | `background` | `bg="background"` |
| Base panel / section | `surface` | `bg="surface"` |
| Default card / dialog | `surface/container` | `bg="surface.container"` |
| Subtle input background | `surface/container/low` | `bg="surface.container.low"` |
| Elevated card / nav | `surface/container/high` | `bg="surface.container.high"` |
| Chip / tag bg | `surface/variant` | `bg="surfaceVariant"` |
| Tooltip / snackbar | `inverse/surface` | `bg="inverseSurface"` |
| Dialog overlay | `scrim` | `bg="scrim"` at reduced opacity |
| Primary CTA | `primary` | `colorPalette="primary"` on `<Button>` |
| Destructive action | `error` | `colorPalette="error"` on `<Button>` |

### Borders

| Purpose | Token | Panda CSS |
|---|---|---|
| Default input / card border | `outline/variant` | `borderColor="outline.variant"` |
| Divider / separator | `outline/variant` | `borderColor="outline.variant"` |
| Strong / focus border | `outline` | `borderColor="outline"` |

### Spacing

| Context | Token | Panda CSS |
|---|---|---|
| Default card padding | `spacing/md` | `p="md"` |
| Section / dialog padding | `spacing/lg` | `p="lg"` |
| Tight internal gap | `spacing/sm` | `gap="sm"` |
| Between-element gap | `spacing/md` | `gap="md"` |
| Icon-to-text nudge | `spacing/xs` | `gap="xs"` |

### Corner radius

| Context | Token | Panda CSS |
|---|---|---|
| Buttons / inputs | `radii/small` | `rounded="sm"` |
| Cards / dialogs | `radii/medium` | `rounded="md"` |
| Pill / toggle / avatar | `radii/full` | `rounded="full"` |

### Legacy 100–900 scale (Chakra / Tailwind convention)

Older design files sometimes used a `Primary/500` naming convention (100–900 scale) instead of M3's 0–100 scale. Approximate mapping:

| Legacy name | M3 Primitive | Semantic token | Light hex |
|---|---|---|---|
| `Primary/500` | `primary/50` | `primary` | `#4C662B` |
| `Primary/400` | `primary/40` | — | `#3F6900` |
| `Accent/500` | `secondary/50` | `secondary` | `#586249` |
| `Stone/60%/500` | `neutral/50` + 60% opacity | `outline` | `#777771` |
| `Stone/60%/400` | `neutral/40` + 60% opacity | `onSurface.variant` | `#5E5F59` |

> Always rebind to the **Semantic** collection — not Primitives — to get dark mode support.
