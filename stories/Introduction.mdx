## ğŸ—ï¸ Architecture

This design system uses a three-layer architecture:

**Layer 1: Infrastructure** (unchanging)
- TypeScript interfaces and contracts
- Build system (tsup, Storybook, Panda CSS)
- Component logic (Ark UI primitives)

**Layer 2: Design Language** (swappable)
- Token values and semantic mappings
- Currently: Material Design 3
- Easily swap by changing one import

**Layer 3: Component Recipes** (derived)
- Panda CSS recipes with variants
- Automatically adapts to active language
